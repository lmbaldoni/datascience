Base de Datos

Base de datos - SQL

Base de datos esta en todos lados! Aprende a manipular una base de datos con este curso teoríco y práctico

Descripción

Aprende a gestionar bases de datos a través de uno de los manejadores más populares, mySQL. En este curso aprenderás, desde la teoría en que se fundamentan las Bases de Datos hasta su implementación, pasando por el conocimiento del lenguaje estándar SQL.
